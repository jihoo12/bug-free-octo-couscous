- Sigma types (Σ, fst, snd) — you have Pi and Path but no dependent pairs, which severely limits what you can express.

- fill — a common derived form from hcomp, but worth having as a primitive or built-in derived.

- composition for Glue types — the hcomp case doesn't handle TGlue specially, which is required for univalence to compute correctly.
